# DevelopersHub Corporation  
## AI/ML Engineering Internship Tasks

### Internship Program Overview
This repository contains the AI/ML Engineering Internship tasks completed as part of the **DevelopersHub Corporation Internship Program**.  
The objective of this internship is to gain **hands-on experience** in data analysis, machine learning, and AI-based solutions using real-world datasets.

### Dataset Used
- **Iris Dataset** (CSV)

### Models Applied
- No machine learning model applied (Exploratory Data Analysis task)

### Key Results & Findings
- Identified clear separability between flower species based on petal length and width.
- Visualizations helped reveal feature distributions and outliers.
- Gained understanding of dataset structure and feature correlations.

## Task 3: Heart Disease Prediction

### Task Objective
To predict whether a patient is at risk of heart disease based on medical attributes.

### Dataset Used
- **Heart Disease UCI Dataset** (Kaggle)

### Models Applied
- Logistic Regression  
- Decision Tree Classifier

### Key Results & Findings
- Logistic Regression provided stable and interpretable results.
- Important features such as age, cholesterol, and blood pressure influenced predictions.
- Model evaluation was performed using accuracy, ROC curve, and confusion matrix.

## Task 5: House Price Prediction

### Task Objective
To predict house prices based on property features such as size, bedrooms, and location.

### Dataset Used
- **House Price Dataset** (Kaggle)

### Models Applied
- Linear Regression  
- Gradient Boosting Regressor

### Key Results & Findings
- Gradient Boosting achieved better prediction accuracy than Linear Regression.
- Feature scaling improved model performance.
- Evaluation using MAE and RMSE highlighted prediction reliability.
